Headhunter is a reverse shell (C2) Command and Control Server. 
Headhunter also has functionality to generate payloads for GNU Linux and Windows 64
and 32 bit operating systems. Note that a standard netcat reverse shell with the
-e flag is compatible with Headhunter. 

## Extra Notes
Currently the Windows payload is still in development but Headhunter works perfectly fine with most reverse shell payloads, including netcat and reverse shells written in python3.
